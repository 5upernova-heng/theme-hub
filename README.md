## Mihomo Party 主题

> --heroui-xxx 变量必须是hsl色值

```css
/* 主题名 */

/* 浅色 */
.light, [data-theme="light"] {
  --heroui-primary: 331.85 69.23% 77.06% !important;
  --heroui-secondary: 157.58000000000004 47.15% 62.16% !important;
}

/* 深色 */
.dark, [data-theme="dark"] {
  --heroui-primary: 331.85 69.23% 77.06% !important;
  --heroui-secondary: 157.58000000000004 47.15% 62.16% !important;
}
```
